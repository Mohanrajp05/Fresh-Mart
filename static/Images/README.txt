To add the logo to your website:

1. Save the green shopping basket logo image as "logo.png" in this directory (FreshMarketPlace/static/Images/)
2. Make sure the file is named exactly "logo.png" (case-sensitive)
3. The image should be in PNG format for best results
4. Recommended size: around 200x200 pixels

The logo will then appear in the navigation bar of your website.
